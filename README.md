# Калькулятор вкладов

Веб-приложение для расчета доходности вкладов и активов, построенное на Next.js с использованием TypeScript и Tailwind CSS.

## Функциональность

- Калькулятор вкладов с учетом капитализации
- Тест заемщика
- Калькулятор доходности активов

## Технологии

- Next.js 14
- TypeScript
- Tailwind CSS
- React

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Запуск

```bash
npm start
```
